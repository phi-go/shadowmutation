# Shadow Mutations

The `bank.py` is the original source file, and `bank_mut.py` is the processed
file with shadow mutations.

```
$ python3 ./bank_mut.py
0 True
1.1 False
2.1 False
2.2 False
3.1 False
3.2 False
```

The `0` is the main line (no mutations). Others are result from each indicated mutants.
